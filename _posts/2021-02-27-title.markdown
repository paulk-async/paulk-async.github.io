# 요약
빈도수가 높은 패턴에 작은 크기의 부호(코드)를 할당하는 방식.
보통 패턴의 단위는 문자(1Byte, 8bit).

# 부호의 생성 방식
이진 트리를 기반으로 함. 저비용의 부호를 만드는 것이므로.

# 부호 간 관계
왜 이런 위와 같은 부호화를 하는 것일까?
빈도수가 높은 패턴(문자)에 0부터 순차적으로 부호를 할당하면 안되는 것일까? -> 디코딩.
